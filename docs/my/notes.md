## Отладочный запуск (Web)

Для запуска приложения в режиме отладки через встроенный веб-сервер выполните команду:

```bash
flutter run -d web-server --web-port 8080 --web-hostname 192.168.2.17  --web-launch-url https://dev.sntralix.ru
```
```bash
flutter run -d web-server --profile --web-port 8080 --web-hostname 192.168.2.17 --web-launch-url https://dev.sntralix.ru
```
